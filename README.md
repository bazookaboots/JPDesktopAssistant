# Electron Demo

## Installation

This project uses [Node.js](https://nodejs.org/en/download/) in order to run the Electron GUI you will probably need to install [Node.js](https://nodejs.org/en/download/) on your machine

## Run Electron GUI

```
cd electron-gui
npm install -d
npm start
```
